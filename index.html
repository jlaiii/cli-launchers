What the launchers handle for you

🔍 Prerequisite Check

Detects Node.js, npm, Codex CLI / Claude Code, and Ollama. Tells you what is missing.

⬇️ Auto-Install

Missing Node.js? It installs it. Missing Ollama? It installs it. Missing the CLI? It installs it.

🔄 Update Checker

Checks npm and GitHub releases for newer versions. One menu press updates everything.

🤖 Model Picker

Browse the top 10 newest cloud models, browse your local models, or type one manually.

🔐 Auth Check

Verifies you are signed into Ollama (`ollama list`). Offers to run `ollama signin` if not.

🚀 One-Click Launch

Builds the right command and starts the Ollama server automatically if needed.

How to use

  1. Download one of the .bat files above (right-click → Save As, or click the green button).
  2. Double-click the file or run it from Command Prompt / PowerShell.
  3. The launcher shows a status screen and menu. Pick an option by typing the number and pressing Enter.
  4. To launch directly without the menu, pass arguments:
    Codex-Launcher.bat launch --model kimi-k2.6:cloud -- --yolo
    Claude-Ollama-Launcher.bat launch --model ollama/llama3
