What is this?

CLI Launchers are single Windows batch files that set up and run the best AI coding assistants available — OpenAI Codex CLI and Anthropic Claude Code — through Ollama.

Instead of manually installing Node.js, npm, the CLI tools, and Ollama, then configuring models and launch flags, you download one .bat file, double-click it, and the launcher handles everything.

Who is it for? Windows developers who want to use cutting-edge AI CLI tools without spending time on setup, dependency hell, or keeping tools up to date. If you want to run Codex or Claude with any Ollama model — cloud or local — this gets you there in one click.

What the launchers handle for you

Prerequisite Check

Detects Node.js, npm, Codex CLI / Claude Code, and Ollama. Tells you what is missing.

Auto-Install

Missing Node.js? It installs it. Missing Ollama? It installs it. Missing the CLI? It installs it.

Update Checker

Checks npm and GitHub releases for newer versions. One menu press updates everything.

Model Picker

Browse the top 10 newest cloud models, browse your local models, or type one manually.

Auth Check

Verifies you are signed into Ollama. Offers to run sign-in if not.

One-Click Launch

Builds the right command and starts the Ollama server automatically if needed.

How to use

  1. Download one of the .bat files above by clicking the green button.
  2. Double-click the file or run it from Command Prompt / PowerShell.
  3. The launcher shows a status screen and menu. Pick an option by typing the number and pressing Enter.
  4. To launch directly without the menu, pass arguments:
    Codex-Launcher.bat launch --model kimi-k2.6:cloud -- --yolo
    Claude-Ollama-Launcher.bat launch --model ollama/llama3
